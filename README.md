Comparative Analysis of Spectral Clustering Algorithms for Community Detection in Social Networks

This project analyzes different spectral clustering approaches for community detection in social networks. The implemented algorithms include Normalized Symmetric Laplacian, Random Walk Laplacian, and Unnormalized Laplacian based spectral clustering methods.

The algorithms are evaluated on standard benchmark datasets such as the Dolphins Social Network Dataset and Zachary's Karate Club. Performance is analyzed using metrics such as Normalized Mutual Information (NMI) and Modularity.

Additionally, synthetic networks generated using the LFR Benchmark Graph are used to evaluate the scalability and performance of the algorithms as the network size increases.

Experimental results show that the Normalized Symmetric Laplacian spectral clustering algorithm consistently performs better, followed by the Unnormalized Laplacian, while the Random Walk Laplacian method performs comparatively weaker in this implementation.
